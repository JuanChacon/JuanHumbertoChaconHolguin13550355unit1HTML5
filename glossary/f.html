Fetch

The Fetch API provides a JavaScript interface for accessing and manipulating parts of the HTTP pipeline, such as requests and responses. It also provides a fetch () global method that provides an easy and logical way to obtain resources asynchronously over the network.

FindIndex

The FindIndex() method performs the one-time function for each element present in the array:

  • If it find an array element, where the function returns a true value, FindIndex() returns the index of that element in the array (and does not check the remaining values).
  • Otherwise it returns undefined
  • FindIndex() does not execute the array element function without values.

ForEach()

The forEach() method calls a provided function once for each element in an array, in order. Note: forEach() does not execute the function for array elements without values.
