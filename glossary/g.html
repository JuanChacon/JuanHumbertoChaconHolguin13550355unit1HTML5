GetUserMedia

It asks the user permission to use a multimedia device such as a camera or microphone. If the user grants this permission, the successCallback is invoked in the called application with a LocalMediaStream object as an argument.

GroupCollapsed()

Creates a new inline group in the Web Console. Unlike console.group(), however, the new group is created collapsed. The user will need to use the disclosure button next to it to expand it, revealing the entries created in the group.
